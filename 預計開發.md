🏡 日常生活類
	1.	AutoOrganizer（自動整理員）
每晚掃描桌面與下載資料夾，把文件、圖片、自動分類進對應資料夾，並產生一份簡報式的「今日檔案摘要」。
→ 可用 Python + AppleScript 實作，配合 Spotlight index。
	2.	Habit Whisper（習慣追蹤員）
用日曆與提醒事項整合，觀察你每天的打開 App、打字時間、螢幕使用紀錄，生成「注意力報告」與小建議。
→ 可透過 Screen Time API + Shortcut 自動化。
	3.	Expense Watcher（消費小記）
從郵件或 LINE Pay 通知抓出金額，自動歸類消費，月底生成簡報圖表。
→ 用 Mail parsing + Numbers/CSV 自動輸出。
	4.	Voice Butler（語音執行員）
結合 Speech Recognition API，讓你用一句話控制電腦（開程式、啟動自動整理、轉檔、開投影片等）。
→ AppleScript + Whisper local model + Shortcuts。

🎓 學校學習類
	1.	Lecture Summarizer（課堂筆記整理員）
你錄課或上線上課後，它自動轉錄成文字、摘要出重點、整理成 Word 筆記或 Notion。
→ Whisper + GPT 本地處理 + Python docx。
	2.	Assignment Tracker（作業管理員）
掃你的郵件、Teams、Moodle，建立所有作業清單、期限倒數，整合提醒。
→ 可用 Gmail API + Calendar API。
	3.	Research Buddy（文獻助手）
拖入 PDF，它幫你生成摘要、找出研究問題、列出引用格式（APA/MLA），還能產出一份「快速讀書報告」。
→ GPT + PyMuPDF + Zotero API。
	4.	Study Timer（專注時鐘）
Pomodoro 模式，但會根據你目前課業壓力（行事曆、代辦）自動調整休息長度。
→ SwiftUI + Core Data + Shortcuts。
