## 🚀 立即體驗

### 啟動方式

**Linux / Mac:**
```bash
./run_webapp.sh
```

**Windows:**
```cmd
run_webapp.bat
```

**手動啟動:**
```bash
source .venv/bin/activate
streamlit run graduate_agent/webapp/streamlit_app.py
```

### 首次使用流程

1. **啟動應用**
   ```bash
   ./run_webapp.sh
   ```

2. **配置帳號**（⚙️ 設定頁面）
   - 填入 Moodle 帳號密碼
   - 點擊「💾 儲存設定」

3. **爬取課程**（📚 爬取課程頁面）
   - 點擊「🚀 開始爬取」
   - 觀看即時進度

4. **下載資源**（📥 下載資源頁面）
   - 選擇要下載的課程
   - 點擊「📥 開始下載」

5. **查看結果**（📊 狀態查看頁面）
   - 檢視統計資訊
   - 確認所有檔案

## 💡 使用技巧

### 1. 安全設定

不要在 Web UI 中長期儲存密碼，使用環境變數：

```bash
# 創建 .env 檔案
echo "MOODLE_USERNAME=你的學號" > .env
echo "MOODLE_PASSWORD=你的密碼" >> .env

# Web UI 會自動載入
```

### 2. 選擇性下載

只想下載特定課程：
1. 前往「📥 下載資源」頁面
2. 取消勾選不需要的課程
3. 只會下載選中的課程

### 3. 查看詳細錯誤

如果出現錯誤：
1. 展開「查看詳細錯誤」區塊
2. 複製錯誤訊息
3. 檢查配置或網路連線

### 4. 定期更新

每週使用流程：
1. 開啟 Web UI
2. 爬取最新課程資料
3. 下載新資源（自動跳過已有檔案）
4. 查看狀態頁面確認

## ❓ 常見問題

**Q: Web UI 的配置會保存嗎？**
A: 配置保存在 session 中，關閉瀏覽器後會消失。建議使用 .env 檔案。

**Q: 可以同時執行多個任務嗎？**
A: Streamlit 是單執行緒的，一次只能執行一個任務。

**Q: 如何停止 Web 應用？**
A: 在終端機按 `Ctrl+C`

**Q: 支援行動裝置嗎？**
A: Streamlit 有基本的響應式設計，但最佳體驗是桌面瀏覽器。

**Q: 可以部署到雲端嗎？**
A: 可以！參考 [WEBAPP_ROADMAP.md](WEBAPP_ROADMAP.md) 中的部署方案。
